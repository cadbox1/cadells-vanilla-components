# Cadell's Vanilla Components

## Commands

- `yarn start` - start rollup dev server.
- `yarn storybook` - start storybook. run this in another terminal, in addition to yarn start.
- `yarn analyze` - bundle size stats.
- `yarn test` - run the rests.
